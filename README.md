Gatsby front end for Third and Grove's Drupal backend.

## Installation

```shell
yarn
```

copy the `sample.env` to `.env` and add the appropriate credentials.

## Development

```shell
yarn start
```

## Build

```shell
yarn build
```

## Testing

ensure that you have your dev server running `yarn start`

for developing tests use -

```shell
yarn cypress
```

for testing alone -

```shell
yarn test
```
